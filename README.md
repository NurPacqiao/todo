This project moves beyond simple scripting by implementing SOLID principles:

Single Responsibility Principle (SRP):

Task class: Handles data structure.

StorageManager class: Handles localStorage operations.

UIManager class: Handles DOM rendering and updates.

Separation of Concerns: The logic (JS) is decoupled from the styling (CSS/Bootstrap) and the structure (HTML).
